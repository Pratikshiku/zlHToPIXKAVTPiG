## Community Elderly Care Service Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/175678d731444c12984e6697519beeaa/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Service Information
6. Service Application
7. Announcement Information
8. User Information Modification Review


**Administrator Functions:**
1. Login
2. Personal Center
3. Administrator Management
4. Service Type Management
5. Announcement Type Management
6. Service Information Management
7. Service Application Management
8. Announcement Information Management
9. User Management
10. User Information Modification Review Management
